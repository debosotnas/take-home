# Take Home Project
Take Home is an interview project for Hero.

## Installation
Use node package manager to install Take Home dependencies [npm](https://www.npmjs.com/).

```bash
npm install
```

## Run project

```javascript
npm start
```
## Build project to DIST folder

```javascript
npm run build
```

# Main dependencies
- Webpack
- Babel
- Dom-Chef
- Sass
