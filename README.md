# Take Home Project

Take Home is an interview project for Hero.

## Installation

Use node package manager to install Take Home dependencies [npm](https://www.npmjs.com/).

```bash
npm install
```

## Run project

```javascript
npm start
```
## Build project dist

```javascript
npm run build
```
